InventoryPro – Stock Management System

Tech Stack
- Angular 12
- .NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

 Features
- Secure Login (JWT)
- Product CRUD operations
- Stock Management (IN / OUT transactions)
- Real-time Stock Calculation
- Transaction History Tracking
- Dashboard with analytics
- Search, Filter, Pagination

Highlights
- Implemented transaction-based inventory system
- Used JWT authentication for secure APIs
- Designed scalable backend with service layer
- Built responsive UI with Bootstrap

Dashboard
- Total Products
- Total Stock
- Low Stock Alerts

 How to Run
Backend
- Run API project
- Apply migrations

### Frontend
- npm install
- ng serve
